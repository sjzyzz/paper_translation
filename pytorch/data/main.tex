Pytorch的数据加载工具核心是\lstinline{torch.utils.data.DataLoader}类。它提供一个遍历数据集的iterable，并支持
\begin{itemize}
    \item 映射风格和迭代风格的数据集
    \item 自定义数据加载顺序
    \item 自动形成批
    \item 单进程多进程数据加载
    \item 自动内存固定
\end{itemize}
这些选项通过DataLoader的构建函数参数设置，它的签名是：
\begin{lstlisting}[language=Python]
DataLoader(
    dataset, 
    batch_size=1, 
    shuffle=False, 
    sampler=None, 
    batch_sampler=None, 
    num_worker=0, 
    collate_fn=None,
    pin_memory=False, 
    drop_last=False, 
    timeout=0, 
    worker_init_fn=None,
    *, 
    prefetch_factor=2, 
    persistent_workers=False
)
\end{lstlisting}
