自建的dir()函数被用来查看模块定义的名字。它返回一个有序的字符串列表：
\begin{lstlisting}[language=bash]

如果不提供参数，dir()将会展示你现在已经定义的名字：
\begin{lstlisting}[language=bash]
    >>> a = [1, 2, 3, 4, 5]
    >>> import fibo
    >>> fib = fibo.fib
    >>> dir()
    ['__builtins__', '__name__', 'a', 'fib', 'fibo', 'sys']
\end{lstlisting}
注意它展示了所有类型的名字：变量、模块、函数等等。
